# BRITHA

A cinematic, immersive digital universe built with Next.js, TypeScript, Tailwind, Framer Motion, Prisma, and NextAuth.

## Quick start

1. Install dependencies
   - npm install
2. Create a PostgreSQL database named britha
3. Set DATABASE_URL in .env.local
4. Run Prisma migrate
   - npx prisma migrate dev --name init
5. Seed demo content
   - npx prisma db seed
6. Start the app
   - npm run dev
